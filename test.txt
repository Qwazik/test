1) Написать в рамках одного класса механизм для работы с базой данных MySQL и для работы с Redis на PHP.

MySql методы:
а) Подключение к БД с возможностью задания параметров подключения как стандартных, так и уникальных.
б) Запрос к БД.
в) Подсчет кол-ва строк, которые вернул запрос.
г) Преобразование результата запроса в ассоциативный массив.
д) Закрытие соединения к БД.

Redis методы:
а) Подключение аналогично с MySQL.
б) Запись значения.
в) Чтение значения.
г) Установка и проверка срока жизни записи.

Прокомментировать основные действия.

2) Получить данные с помощью Ajax запроса из файла data.json и вывести данные, на основе блочной верстки, как показано на рисунке "пример.png".
Показанные на рисунке параметры находятся в узле Events. "C" - коэффициент, "G" - id группы событий, "P" - параметр который указан в скобках в названии ставки, "T" - id типа ставки.
Сопоставления числовых id групп и типов ставок с текстовыми названиями находятся в файле betsNames_ru.js.

После вывода данных групп ставок навесить обработчки для добавления выбранного события в купон. Пример купона показан в файле купон.png. В купоне должна быть возможность удалять добавленное событие. Рассчет суммы выигрыша вычисляется по формуле сумма ставки * на коэффициент. Коэффициент считается путем умножения все коэффициентов купона.
